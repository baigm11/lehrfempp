Detailed Description

template<class F>
class lf::uscalfe::MeshFunctionGlobal< F >

MeshFunction wrapper for a simple function of physical coordinates.

Template Parameters
Fa functor type, offering an evaluation operator that acccepts a single coordinate vector.

This class is meant ot create MeshFunction compliant objects that wrap a function \(d:\Omega\mapsto X\) on the physical domain. The image type of the function can be anything at this point.
